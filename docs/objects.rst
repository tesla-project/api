JSON Objects
=============

.. json:object:: activity

    :property integer vle_id: VLE identifier
    :property string activity_id: activity identifier
    :property string activity_type: activity type

.. json:object:: audit_data

    :property integer include_enrolment: Enrollment samples must be included by TEP to the provided data (1) or not (0)
    :property integer include_request: Orignial request must be included by TEP to the provided data (1) or not (0)
    :property string audit_data: String with a JSON structure with the audit data generated by the instrument

.. json:object:: sample_data

    :property string mime_type: mime_type of the data contained in this JSON object
    :property string file_name: the filename provided by VLE for this data (optional)
    :property string b64_data: the data encoded in base64 format
